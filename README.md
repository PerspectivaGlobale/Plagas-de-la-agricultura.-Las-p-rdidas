# Plagas-de-la-agricultura.-Las-pérdidas
Entre los problemas que el agricultor debe resolver con el auxilio de la tecnología figura el de las plagas que pueden afectar directamente a los cultivos y a los animales en pie, o bien a los productos almacenados.
En el primer caso, se produce una disminución del rendimiento; en el segundo, una merma en el volumen o en la calidad del producto ya cosechado. No es extraño que cuando se trabaja mal se sumen ambos fenómenos.
